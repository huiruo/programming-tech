
## object-cover object-contain object-fil的区别
```css
<img
  src={iconUrl}
  alt={item.game_name}
  // className="h-[62px] w-[100px] rounded-md object-cover"
  // className="h-[62px] w-[100px] rounded-md object-contain"
  className="h-[60px] w-[70px] rounded-md object-fill"
/>
```

它们影响的是 图片（或视频）在固定尺寸容器中如何缩放与裁剪。

### 1.object-cover
形象比喻： 像一张照片裁进相框，宁可切掉边缘，也不让照片变形。
1. 让图片完全填满容器，同时保持比例，不变形；
  - 但如果比例不同，会有部分图片被“裁掉”。

2. 效果：
  - 图片会放大到刚好覆盖容器的全部区域；

  - 如果图片比例 ≠ 容器比例 → 会被裁切；

  - 常用于 头像、banner、缩略图。

### object-contain
形象比喻： 像是把整张照片缩小塞进相框，宁可留白，也不裁边。

1. 完整显示整张图片，保持比例，不裁切；
如果比例不同，会有空白（留白区）。

2. 效果：
  - 图片会被等比缩放，完全显示；
  - 容器背景可能会露出来；
  - 常用于 logo、图标、插画（不能被裁切）。

### object-fill
形象比喻： 像在 Word 里拉伸图片，宽高都强行适配。

强制拉伸图片填满容器，不保持比例，会变形。

1. 效果：
  - 图片完全占满容器；
  - 比例失真；
  - 适合非关键图片、背景纹理等。
