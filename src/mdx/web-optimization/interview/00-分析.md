## test
```
// className="rounded-xl h-auto object-cover"
className="object-contain h-full w-auto" 二者区别
```
* 游戏封面、海报、展示图	object-cover h-auto
* 图标、徽章、带透明背景图像	object-contain h-full w-auto

| 属性       | `object-cover` + `h-auto` | `object-contain` + `h-full w-auto` |
| -------- | ------------------------- | ---------------------------------- |
| 填充方式     | 裁剪并填满容器                   | 保留完整比例，缩放到容器内                      |
| 是否可能裁剪图片 | ✅ 可能（超出部分被裁）              | ❌ 不会裁剪                             |
| 是否可能留白   | ❌ 通常会填满                   | ✅ 图片比例不一致会留白                       |
| 典型用途     | 做头像、背景、卡片封面图              | 显示图标、LOGO、按钮图等                     |
| 高度       | 图片高度自动适应宽度                | 图片高度总是充满容器高度                       |
| 宽度       | 自动，随内容缩放                  | 宽度自动，保持比例                          |
